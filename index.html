4、向量点乘

结果是一个值

给两个向量，可以算出夹角余弦

单位向量的点乘就是夹角余弦

性质

交换律、分配律、结合律

运算

点乘就是对应的元素相乘再相加

作用

1、计算两个向量的夹角
2、求一个向量在另一个向量上的投影

由点乘可以推出夹角余弦，继而推出投影

3、将一个向量分为水平和竖直两个向量

三维向量，拆分成以uvw为坐标系的分量

4、重要：计算在正面还是背面

以A向量为基准

B向量与A向量点乘结果>0 cos在0-90度大于0

C向量与A向量点乘结果<0 cos在90-180小于0

方向完全相同，点乘为1 cos0 = 1

方向完全相反，点乘为-1 cos180 = -1

5、向量叉乘

结果是一个向量

叉乘结果垂直于叉乘的两个向量所在的平面

注意：a x b = -b x a 可以用右手法则来理解，叉乘结果始终向上

性质

不满足交换律

运算

作用

1、可以用于建立一个三维空间的坐标系

叉乘自己，得到的结果是0向量，因为sin0 = 0

2、判断左右/判断内外

观察叉乘的结果方向，判断左右

同理可以判断点是否在三角形内部

编译与链接

发表于 2025-10-26 分类于 C++工程管理 本文字数： 321 阅读时长 ≈ 1 分钟

一、什么是CMake？

CMake是一个跨平台的编译工具，可以用简单的语句来描述所有平台的编译链接过程

二、VisualStudio工程

使用VisualStudio创建的工程，属于windows下，并且只能够用相同的IDE打开

三、编译与链接

编译：将当前的C++代码通过编译器，编译成为目标代码的过程

链接：将多段编译好的目标代码，互相进行链接，形成一个完整的可运行程序(exe)

四、静态链接/动态链接

静态链接库：包含代码与数据的文件，可在程序编译期间被链接入程序(.lib/.o)

动态链接库：包含代码与数据的文件，可在程序执行期间，被动态加载
